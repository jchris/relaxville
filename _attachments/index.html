Welcome to Relaxville!

This page lists various views you can get of the test data in this database, as well as actions you can take to share your results.

If you are running this application on your own CouchDB instance, the views are only over the test runs that are on your local machine. Reports are contributed by replicating your test_suite_reports database to the database at http://couchdb.couchdb.org/test_suite_reports.

Click this link to contribute your test run data for everyone to browse.

The closest thing to identifying data in the report documents is your timezone and the version of CouchDB you are using. You can view the report documents yourself to be sure you don't send any data to the project that you'd like to keep private. We think the reports are pretty safe to share, hopefully you agree. Please contact the project if you have any reservations, so we can help put them to rest.
